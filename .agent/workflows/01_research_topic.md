   - 実行対象のフォルダ名（例：`grade2/unit_06/01_比較級・最上級`）から、学年と単元テーマを特定する。

2. **Web検索 (Research)**
   - `search_web` ツールを使用し、特定したトピックについて以下の情報を調査する。
     - 学習指導要領における範囲（必須項目）
     - 主要教科書（NEW HORIZONなど）で扱われる単語・表現
     - 高校レベルとして除外すべき内容（発展的な用法など）
   - 検索クエリ例：「中学2年生 英語 比較級 最上級 学習指導要領 範囲」「中学英語 比較級 必須単語」

3. **構成案作成**
   - 「1ファイル1内容」の原則に基づき、必要なファイル構成を計画する。
   - 例：
     - `topic1.md`: 規則変化
     - `topic2.md`: 長い語 (more/most)
     - `topic3.md`: 不規則変化

4. **ファイル作成・更新**
   - `topic_rules.md` (本ファイル下部) に従い、各 `topic*.md` ファイルを作成または更新する。
   - 検索結果に基づき、正確かつ中学生に適したレベルで記述する。

5. **自己チェック**
   - 作成したファイルの内容が、ルール（ファイル名、フォーマット、レベル感）に適合しているか確認する。
   - ユーザーにレビューを依頼する準備をする。

# 補足資料

## topic.md 作成ルール
`topic.md` ファイルを作成・更新する際は、以下のルールに従ってください。

### 1. ファイル形式
- ファイル名は必ず `topic.md` または `topic[数字].md`（例：`topic1.md`）とする。
- フォーマットは Markdown 形式とする。
- 文字コードは UTF-8 とする。

### 2. コンテンツ構造
- **タイトル**: `# [トピック名] のポイント` または `# [トピック名] の詳細ポイント` とする。
- **本文**: 箇条書き (`-`) を使用して記述する。
- **階層**: 必要に応じてインデントを使用し、階層構造を作る。
- **原則**: 1つのファイルで扱う主要な内容は1つに絞る。密接に関連する項目はまとめてもよいが、変化のルールや用法が大きく異なる場合はファイルを分けることが望ましい。
- **対応関係**: `video_content.md`（または `video_content[数字].md`）と対になるように作成し、扱うトピックを一致させる。

### 3. 記述内容のレベル
- **中学1年生〜中学2年生 (Unit 5まで)**:
  - 基本的な文法事項、意味、簡単な使い方に留める。
  - 簡潔な箇条書きとする。

- **中学2年生 (Unit 6以降) 〜 中学3年生**:
  - 詳細な解説を含める。
  - 文法ルール、具体的な例文、否定文・疑問文の作り方、関連する慣用表現などを網羅する。
  - 定期テストや入試で問われやすいポイント（書き換え、例外など）も含める。

### 4. 重複の回避
- 複数のフォルダにまたがるトピック（例：比較級と as...as）は、それぞれのフォルダの主題に合わせて内容を分割する。
- 親トピック（例：比較級・最上級）のファイルには、別フォルダ（例：as...as）で扱われている詳細内容は含めず、参照または概要に留めるか、完全に分離する。

### 5. 学習指導要領・教科書範囲の確認
- 中学英語の範囲（検定教科書レベル）を逸脱する表現（例：倍数表現、高校レベルの慣用句など）は、原則として含めない。
- 「発展」や「参考」として扱う場合は、その旨を明記するか、別セクションに分ける。
- **網羅性チェック**: その単元で必ず教えるべき重要構文・用法が漏れていないか確認する。
- **過剰性チェック**: 公立高校入試レベルを超えた難解な単語や構文が含まれていないか確認する。
- 迷う場合は、学習指導要領や主要な教科書（NEW HORIZON等）のカリキュラムを確認する。

## NEW HORIZON 学年別 英文法一覧

### 中学1年生 (NEW HORIZON 1)
| Unit | 文法項目 | 概要・教科書の記述 |
| :--- | :--- | :--- |
| **Unit 1** | 一般動詞とbe動詞 | I am ..., I like ... などの一般動詞の文 / 疑問文・否定文 |
| **Unit 2** | 代名詞とcan | 主語に代名詞を使うbe動詞の文 / can を使う文 / this, that |
| **Unit 3** | 疑問詞 (1) | what, who, how, where, when を使う疑問文 |
| **Unit 4** | 複数形・命令文 | 名詞の複数形 / how many / what time / 命令文・否定の命令文 |
| **Unit 5** | 三人称単数現在形 | 三単現のs (肯定・疑問・否定) |
| **Unit 6** | 代名詞 (2)・疑問詞 (2) | 目的語に使う代名詞 (him, her等) / whose, which を使う疑問文 |
| **Unit 7** | 現在進行形・感嘆文 | 現在進行形 (肯定・疑問) / 感嘆文 (What/How) |
| **Unit 8** | 不定詞 (1)・look | 不定詞 (名詞的用法) / look + 形容詞 (〜に見える) |
| **Unit 9** | 一般動詞の過去形 | 規則動詞・不規則動詞 / 過去の疑問文・否定文 |
| **Unit 10** | be動詞の過去・過去進行形 | be動詞の過去形 / 過去進行形 / There is[are] ... の文 |
