演算処理をしていない回路への\n電源を遮断する技術は？
パワーゲーティング
リーク電流による不要な電力消費を防ぐ省電力技術です。

3Dの画像処理を高速に行う\n専用の装置を何という？
GPU
数千個のコアによる並列処理が得意で、AI学習にも使われます。

GPUが特に得意としている\n演算の種類は？
行列演算
画像描写やディープラーニングで必要となる計算です。

電源を切ると内容が\n消えてしまう特性を何という？
揮発性
RAMなどが持つ性質で、通電中のみデータを保持します。

電源を切っても内容を\n保持し続ける特性を何という？
不揮発性
ROMやフラッシュメモリなどが持つ、データが消えない性質です。
