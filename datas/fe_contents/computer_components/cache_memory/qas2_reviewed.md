主記憶を複数の区画に分け\n並列アクセスする技術は？
メモリインタリーブ
バンクと呼ばれる単位に分け、連続データの転送を速めます。

エラー訂正機能を持つ\nメモリの呼び名は？
ECCメモリ
データの誤りを検出し、自動的に修復できるメモリです。

ECCメモリで使われる\n誤り訂正符号の名前は？
ハミング符号
1ビットの誤りを訂正し、2ビットの誤りを検出可能です。

紫外線を使ってデータの\n一括消去を行うROMは？
UV-EPROM
専用の装置で光を当てることで、何度でも書き換えられます。

電気的にデータの消去・\n書き換えができるROMは？
EEPROM
電圧をかけることで、バイト単位での部分消去が可能です。
