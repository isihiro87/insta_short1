CPUと主記憶の速度差を\n吸収するためのメモリは？
キャッシュメモリ
高速なSRAMを使用し、CPUの待ち時間を減らします。

キャッシュメモリに\nデータが存在する確率を何という？
ヒット率
この値が高いほど、高速なメモリから読み出せる効率が上がります。

データが主記憶に存在する確率は？
NFP（Not Found Probability）
1からヒット率を引いた値で、主記憶へのアクセス頻度を示します。

キャッシュと主記憶の両方に同時に書き込む方式は？
ライトスルー方式
データの一貫性は保たれますが、書き込み速度は低速です。

キャッシュのみ書き込み、後で主記憶へ反映する方式は？
ライトバック方式
主記憶へのアクセス回数が減るため、処理が高速になります。
