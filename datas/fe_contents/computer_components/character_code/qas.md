文字を識別するための固有の番号を何という？
文字コード
2進数の数字をそれぞれの文字に割り当てた識別番号です。

世界中の文字を一つの体系にしたコードは何？
Unicode
国や言語を問わず共通で利用できる国際的な文字コードです。

アメリカ発祥の最も基本的な文字コードは何？
ASCIIコード
英数字、記号、制御文字のみを収録した基本コードです。

ASCIIに漢字・仮名を加えた日本語コードは？
シフトJISコード
日本で広く普及している代表的な日本語文字コードです。

UNIXやLinuxで使われる日本語文字コードは何？
EUC
日本語の漢字や仮名を扱うことができる文字コードです。

Unicodeの符号化方式で最も普及しているのは？
UTF-8
インターネットの世界で最も標準的に使われる符号化方式です。
