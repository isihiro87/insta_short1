# 受け身 (現在形) 動画コンテンツ

## 1. 最初に提示する例題
**(1) 日本語訳に合うように、空所に適切な語を入れよう。**
このロボットは日本で作られています。
This robot ( is ) ( made ) in Japan.
**正解**: is made

## 2. 読み上げ原稿
第1問！
カッコに入る単語は分かるかな？

### パターンB：型（ルール）から入る
受け身の文を作るときのルールはこれ！
「主語 + be動詞 + 過去分詞」
この型に単語を当てはめていくよ。
主語は This robot だから、be動詞は is だね。
そして「作られる」だから、make の過去分詞 made を入れるよ。
これで「作られています」っていう意味の完成！

## 3. 必要な画像
* 日本の工場で作られているロボットのイラスト
* "People make this robot" から "This robot is made" への変化を示す矢印
* 「主語 + be動詞 + 過去分詞」という枠組みのイラスト
* ロボットにスポットライトが当たっているイラスト
* "This robot made..." にバツがついているイラスト
