# 受け身 (疑問文) 動画コンテンツ

## 1. 最初に提示する例題
**(1) 日本語訳に合うように、空所に適切な語を入れよう。**
このロボットは日本で作られていますか。
( Is ) this robot ( made ) in Japan?
**正解**: Is, made

## 2. 読み上げ原稿
第1問！
カッコに入る単語は分かるかな？


### パターンA：具体例から入る
まずはこの文を見て。
"This robot is made in Japan."
「このロボットは日本で作られています」っていう文だね。
これを「作られていますか？」って聞きたいときは、
be動詞の is を主語の前に出すよ。
"Is this robot made in Japan?"
be動詞の文の疑問文と同じで、be動詞を前に出すだけだね。

### パターンB：型（ルール）から入る
受け身の疑問文のルールはこれ！
「be動詞 + 主語 + 過去分詞 ...?」
この型に当てはめてみよう。
be動詞の Is を一番前に置いて、
その後に主語の this robot、そして過去分詞の made を続けるよ。
最後にクエスチョンマークを忘れずに！

### パターンC：イメージ（気持ち）から入る
疑問文は「聞きたい！」っていう気持ちが前に出るよね。
だから、be動詞の Is を文の先頭にポンと出してあげるんだ。
「Is ...?」って始めることで、「質問だよ！」って合図を送るイメージだよ。

### パターンD：NG例（間違い探し）から入る
よく "Do you made ...?" とか "Does this robot made ...?" って書いちゃうことない？
受け身の文には be動詞がいるから、Do や Does は使わないよ！
「〜されますか？」って聞くときは、必ず be動詞を使ってね。

### パターンE：リズム・暗記から入る
「〜されますか？」は「Is ... 過去分詞？」
Is this robot made? Is this robot made?
リズムで覚えちゃおう。
答え方も "Yes, it is." だよ。

## 3. 必要な画像
* ロボットのイラストに「？」マークがついている画像
* "This robot is made" から "Is this robot made" への語順変化を示す矢印
* 「be動詞 + 主語 + 過去分詞」という枠組みのイラスト
* Is が飛び出しているイメージのイラスト
* "Does this robot..." にバツがついているイラスト
