# 受け身 (否定文) 動画コンテンツ

## 1. 最初に提示する例題
**(1) 日本語訳に合うように、空所に適切な語を入れよう。**
このロボットは日本で作られていません。
This robot ( is ) ( not ) made in Japan.
**正解**: is not

## 2. 読み上げ原稿
第1問！
カッコに入る単語は分かるかな？


### パターンA：具体例から入る
まずはこの文を見て。
"This robot is made in Japan."
「このロボットは日本で作られています」っていう文だね。
これを「作られていません」って否定したいときは、
be動詞の is の後ろに not を入れるよ。
"This robot is not made in Japan."
be動詞の文の否定文と同じで、be動詞の後ろに not を置くだけだね。

### パターンB：型（ルール）から入る
受け身の否定文のルールはこれ！
「主語 + be動詞 + not + 過去分詞」
この型に当てはめてみよう。
be動詞の is のすぐ後ろに not を入れるのがポイントだよ。
is not made で「作られていない」ってなるんだ。

### パターンC：イメージ（気持ち）から入る
否定の not は「違うよ！」って打ち消す言葉だよね。
だから、be動詞の直後に置いて、「そうじゃないんだ」って強く否定するイメージを持とう。
is not made... 「作られて、ない！」って感じだね。

### パターンD：NG例（間違い探し）から入る
よく "This robot don't made ..." とか "doesn't made ..." って書いちゃうことない？
受け身の文は be動詞を使っているから、don't や doesn't は使わないよ！
be動詞の文のルール通り、not を使うんだ。

### パターンE：リズム・暗記から入る
「〜されていない」は「be動詞 + not + 過去分詞」！
is not made, is not made...
短縮形の isn't made でもOKだよ。
リズムよく口ずさんで覚えちゃおう。

## 3. 必要な画像
* ロボットのイラストに「×」マークがついている画像
* "is made" の間に "not" が入る様子を示す図
* 「主語 + be動詞 + not + 過去分詞」という枠組みのイラスト
* not が「違う！」と言っているイメージのイラスト
* "doesn't made" にバツがついているイラスト
