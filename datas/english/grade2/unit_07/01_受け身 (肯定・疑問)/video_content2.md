# 受け身 (過去形) 動画コンテンツ

## 1. 最初に提示する例題
**(1) 日本語訳に合うように、空所に適切な語を入れよう。**
あの学校は先月建てられました。
That school ( was ) ( built ) last month.
**正解**: was built

## 2. 読み上げ原稿
第1問！
カッコに入る単語は分かるかな？


### パターンA：具体例から入る
まずはこの文を見て。
"They built that school last month."
「彼らは先月あの学校を建てました」っていう文だね。
これを「あの学校は建てられました」って言いたいときは、
過去の話だから、be動詞を過去形の was にして、
build の過去分詞の built とセットで使うよ。
"That school was built last month."
これで「建てられた」っていう過去の受け身になるよ。

### パターンB：型（ルール）から入る
過去の受け身のルールはこれ！
「主語 + was/were + 過去分詞」
この型に当てはめてみよう。
主語は That school で単数だから、be動詞は was だね。
そして「建てられた」だから、build の過去分詞 built を入れるよ。
これで「建てられました」の完成！

### パターンC：イメージ（気持ち）から入る
「建てられた」っていうのは、もう終わったことだよね。
だから「過去の印」が必要なんだ。
be動詞を was に変えることで、「昔のことだよ」って伝えてあげるイメージだよ。
was built で「建てられた（過去）」！

### パターンD：NG例（間違い探し）から入る
よく "That school is built last month." って書いちゃうことない？
last month（先月）ってあるのに、is（現在）を使うと変だよね。
「先月」のことだから、be動詞も過去形の was に変えないといけないよ。
時制の一致に気をつけよう！

### パターンE：リズム・暗記から入る
「〜された」は「was/were + 過去分詞」！
was built, was built, was built...
「建てられた」は was built！
リズムよく何回も言って覚えちゃおう。

## 3. 必要な画像
* 新しく建てられた学校のイラスト
* "They built that school" から "That school was built" への変化を示す矢印
* 「主語 + was/were + 過去分詞」という枠組みのイラスト
* カレンダーの「先月」を指差しているイラスト
* "is built" にバツがついているイラスト
