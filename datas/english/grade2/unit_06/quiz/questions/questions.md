# 第1部：比較級（Comparative） 15問

次の文の( )内に入れるのに最も適切なものを、ア～エから1つ選びなさい。

---

Q1. A volleyball is ( ) than a basketball.
「バレーボールはバスケットボールより小さいです。」
ア. smallest　イ. more small　ウ. small　エ. smaller
→ 正解：エ
解説：2つを比べる比較級。smallは短い語なので-erをつけてsmaller。

Q2. Soccer is ( ) popular than tennis in our school.
「私たちの学校ではサッカーはテニスより人気があります。」
ア. most　イ. more　ウ. much　エ. many
→ 正解：イ
解説：popularは長い語なので、前にmoreをつけて比較級にする。

Q3. This question is ( ) than that one.
「この問題はあの問題より簡単です。」
ア. easier　イ. easy　ウ. easyer　エ. more easy
→ 正解：ア
解説：easyはyで終わる語なので、yをiに変えて-erをつける。

Q4. I like spring ( ) than summer.
「私は夏より春が好きです。」
ア. well　イ. best　ウ. better　エ. good
→ 正解：ウ
解説：like ~ better than ...で「…より～が好き」。good/wellの比較級はbetter。

Q5. My bag is not as ( ) as yours.
「私のバッグはあなたのものほど重くありません。」
ア. heavier　イ. heaviest　ウ. more heavy　エ. heavy
→ 正解：エ
解説：not as ~ as ...は「…ほど～ではない」。asとasの間は原級（元の形）。

Q6. Please speak ( ) clearly next time.
「次はもっとはっきり話してください。」
ア. very　イ. more　ウ. much　エ. most
→ 正解：イ
解説：clearlyは副詞。長い語なのでmoreをつけて比較級にする。

Q7. Which is ( ), this red apple or that green apple?
「この赤いリンゴとあの緑のリンゴ、どちらが大きいですか。」
ア. more big　イ. big　ウ. bigger　エ. biggest
→ 正解：ウ
解説：2つを比べる比較級。bigは短い語で子音を重ねて-erをつける。

Q8. This slide is ( ) useful than the old one.
「このスライドは古いものより役に立ちます。」
ア. more　イ. very　ウ. most　エ. many
→ 正解：ア
解説：usefulは長い語なので、前にmoreをつけて比較級にする。

Q9. Ken can run ( ) than Mike.
「健はマイクより速く走ることができます。」
ア. fastest　イ. more fast　ウ. fast　エ. faster
→ 正解：エ
解説：fastは短い語なので-erをつけてfaster。

Q10. Today is ( ) than yesterday.
「今日は昨日より暑いです。」
ア. hottest　イ. hotter　ウ. more hot　エ. hot
→ 正解：イ
解説：hotは短い語で子音を重ねて-erをつける。

Q11. Your presentation was as ( ) as hers.
「あなたの発表は彼女のものと同じくらい良かったです。」
ア. best　イ. well　ウ. good　エ. better
→ 正解：ウ
解説：as ~ as ...は「…と同じくらい～」。asとasの間は原級（元の形）。

Q12. Curling is ( ) interesting than other sports.
「カーリングは他のスポーツよりおもしろいです。」
ア. very　イ. most　ウ. much　エ. more
→ 正解：エ
解説：interestingは長い語なので、前にmoreをつけて比較級にする。

Q13. I got up ( ) than my mother this morning.
「私は今朝、母より早く起きました。」
ア. more early　イ. earliest　ウ. earlier　エ. early
→ 正解：ウ
解説：earlyはyで終わる語なので、yをiに変えて-erをつける。

Q14. This topic is ( ) difficult than yours.
「このトピックはあなたのものより難しいです。」
ア. more　イ. most　ウ. many　エ. very
→ 正解：ア
解説：difficultは長い語なので、前にmoreをつけて比較級にする。

Q15. Writing English is not as ( ) as speaking English.
「英語を書くことは話すことほど難しくありません。」
ア. more hard　イ. hard　ウ. hardest　エ. harder
→ 正解：イ

# 第2部：最上級（Superlative） 15問

次の文の( )内に入れるのに最も適切なものを、ア～エから1つ選びなさい。

---

Q1. This is the ( ) building in our town.
「これは私たちの町で一番古い建物です。」
ア. oldest　イ. old　ウ. most old　エ. older
→ 正解：ア
解説：最上級はthe + 最上級。oldは短い語なので-estをつける。

Q2. Which is the ( ) of the three?
「3つの中でどれが一番大きいですか。」
ア. most large　イ. large　ウ. largest　エ. larger
→ 正解：ウ
解説：of the threeで3つの中から選ぶ最上級。largeは-stをつける。

Q3. I think this movie is the ( ) interesting of all.
「私はこの映画が全ての中で一番おもしろいと思います。」
ア. much　イ. most　ウ. very　エ. more
→ 正解：イ
解説：interestingは長い語なので、前にmostをつけて最上級にする。

Q4. Who runs the ( ) in your class?
「あなたのクラスで誰が一番速く走りますか。」
ア. faster　イ. fast　ウ. most fast　エ. fastest
→ 正解：エ
解説：in your classで範囲を示す最上級。fastは-estをつける。

Q5. August is the ( ) month in Japan.
「8月は日本で一番暑い月です。」
ア. hotter　イ. hottest　ウ. hot　エ. most hot
→ 正解：イ
解説：hotは短い語で子音を重ねて-estをつける。

Q6. He is the ( ) soccer player in this team.
「彼はこのチームで一番上手なサッカー選手です。」
ア. good　イ. well　ウ. best　エ. better
→ 正解：ウ
解説：goodの最上級はbest（不規則変化）。

Q7. This is the ( ) famous picture in the museum.
「これは美術館で一番有名な絵です。」
ア. most　イ. very　ウ. more　エ. best
→ 正解：ア
解説：famousは長い語なので、前にmostをつけて最上級にする。

Q8. Which subject do you like the ( )?
「あなたはどの教科が一番好きですか。」
ア. well　イ. better　ウ. good　エ. best
→ 正解：エ
解説：like ~ the bestで「～が一番好き」。good/wellの最上級はbest。

Q9. Mt. Fuji is the ( ) mountain in Japan.
「富士山は日本で一番高い山です。」
ア. most high　イ. higher　ウ. highest　エ. high
→ 正解：ウ
解説：highは短い語なので-estをつけてhighest。

Q10. This computer is the ( ) useful of all.
「このコンピュータは全ての中で一番役に立ちます。」
ア. more　イ. best　ウ. most　エ. much
→ 正解：ウ
解説：usefulは長い語なので、前にmostをつけて最上級にする。

Q11. My bag is the ( ) of the five.
「私のバッグは5つの中で一番重いです。」
ア. most heavy　イ. heaviest　ウ. heavy　エ. heavier
→ 正解：イ
解説：heavyはyで終わる語なので、yをiに変えて-estをつける。

Q12. This is the ( ) important thing for me.
「これは私にとって一番重要なことです。」
ア. best　イ. more　ウ. very　エ. most
→ 正解：エ
解説：importantは長い語なので、前にmostをつけて最上級にする。

Q13. Which is the ( ) subject for you?
「あなたにとってどの教科が一番難しいですか。」
ア. difficultest　イ. difficult　ウ. most difficult　エ. difficulter
→ 正解：ウ
解説：difficultは長い語なので、前にmostをつけて最上級にする。

Q14. Lake Biwa is the ( ) lake in Japan.
「琵琶湖は日本で一番大きい湖です。」
ア. larger　イ. largest　ウ. most large　エ. large
→ 正解：イ
解説：largeは短い語なので-stをつけてlargest。

Q15. She speaks English the ( ) in our class.
「彼女は私たちのクラスで一番上手に英語を話します。」
ア. best　イ. good　ウ. well　エ. better
→ 正解：ア
解説：wellの最上級はbest（不規則変化）。副詞として使われている。
