### 1. 比較級・最上級 変化リスト

資料の文法解説および問題に登場する形容詞・副詞の変化表です。変化のルールごとに分類しました。

#### 【A】 規則変化（-er / -est をつける語）
基本的な変化をする単語や、語尾が変化するものです。

| 原級 | 意味 | 比較級 (~er) | 最上級 (~est) | 変化のルール |
| :--- | :--- | :--- | :--- | :--- |
| **old** | 古い | **older** | **oldest** | 基本, |
| **small** | 小さい | **smaller** | **smallest** | 基本 |
| **high** | 高い | **higher** | **highest** | 基本 |
| **fast** | 速く | **faster** | **fastest** | 基本 |
| **hard** | 熱心に | **harder** | **hardest** | 基本 |
| **young** | 若い | **younger** | **youngest** | 基本 |
| **light** | 軽い | **lighter** | **lightest** | 基本 |
| **sweet** | 甘い | **sweeter** | **sweetest** | 基本 |
| **cool** | かっこいい | **cooler** | **coolest** | 基本 |
| **large** | 広い、大きい | **larger** | **largest** | eで終わる語は r/st |
| **cute** | かわいい | **cuter** | **cutest** | eで終わる語は r/st |
| **nice** | すてきな | **nicer** | **nicest** | eで終わる語は r/st |
| **big** | 大きい | **bigger** | **biggest** | 重ねて er/est |
| **hot** | 暑い | **hotter** | **hottest** | 重ねて er/est |
| **easy** | 簡単な | **easier** | **easiest** | yをiに変えて er/est |
| **heavy** | 重い | **heavier** | **heaviest** | yをiに変えて er/est |
| **early** | 早く | **earlier** | **earliest** | yをiに変えて er/est |
| **busy** | 忙しい | **busier** | **busiest** | yをiに変えて er/est |

#### 【B】 長い語（more / most をつける語）
つづりの長い形容詞や副詞は、語尾を変えずに前に `more` / `most` を置きます。

| 原級 | 意味 | 比較級 (more ~) | 最上級 (most ~) |
| :--- | :--- | :--- | :--- |
| **famous** | 有名な | **more famous** | **most famous** |
| **popular** | 人気のある | **more popular** | **most popular** |
| **difficult** | 難しい | **more difficult** | **most difficult** |
| **important** | 重要な | **more important** | **most important** |
| **interesting** | おもしろい | **more interesting** | **most interesting** |
| **exciting** | わくわくする | **more exciting** | **most exciting** |
| **useful** | 役に立つ | **more useful** | **most useful** |
| **beautiful** | 美しい | **more beautiful** | **most beautiful** |
| **carefully** | 注意深く | **more carefully** | **most carefully** |
| **clearly** | はっきりと | **more clearly** | **most clearly** |
