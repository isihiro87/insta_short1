# not as ... as 動画コンテンツ

## 1. 最初に提示する例題
**(1) 日本語訳に合うように、空所に適切な語を入れよう。**
この本はあの本ほど面白くありません。
This book is ( not ) ( as ) ( interesting ) as that book.
**正解**: not, as, interesting

## 2. 読み上げ原稿
第1問！
カッコに入る単語は分かるかな？

### パターンA：具体例から入る
まずはこの文を見て。
"This book is as interesting as that book."
「この本はあの本と同じくらい面白い」だね。
これを否定して「〜ほどじゃない」って言いたいときは、
be動詞の後ろに not を入れるよ。
"This book is not as interesting as that book."
後ろの as that book で「あの本ほど〜ない」って比べているんだ。
これで「同じレベルには達していないよ」っていう意味になるんだ。

### パターンB：型（ルール）から入る
「〜ほど...ない」のルールはこれ！
「not as + 形容詞 + as」
as ... as の否定文だね。
not as interesting as ... で「〜と同じくらい面白くはない」
つまり「〜の方が面白い（こっちは負けてる）」ってことだよ。

### パターンC：イメージ（気持ち）から入る
not as ... as は「背伸びしても届かない」イメージ。
「あの本」の面白さレベルに、「この本」は届いていないんだ。
イコール（＝）に not がついて「イコールじゃない（≠）」、
しかも「下回っている（＜）」っていうニュアンスだよ。

### パターンD：NG例（間違い探し）から入る
よく「面白くない」だからって "not interesting" だけにしちゃうことがあるけど、
「あの本ほど」って比べる相手がいるときは as ... as が必要だよ。
not as interesting as ... このセットを忘れないでね。

### パターンE：リズム・暗記から入る
「〜ほど...ない」は「not as ... as」！
not as interesting as, not as big as...
「ノット・アズ・アズ」
このリズムで覚えちゃおう。
「ほどない」は「ノットアズアズ」！

最後に読んでみよう！
This book is not as interesting as that book.
This book is not as interesting as that book.
素晴らしい！

## 3. 必要な画像
* 2冊の本があり、片方（あの本）の方が輝いているイラスト
* as ... as の前に not が入るアニメーション
* 「not as + 原級 + as」という枠組みのイラスト
* 背伸びしても届かないイメージのイラスト
* "not interesting" だけの文にバツがついているイラスト
