# as ... as 動画コンテンツ

## 1. 最初に提示する例題
**(1) 日本語訳に合うように、空所に適切な語を入れよう。**
ケンはトムと同じくらいの背の高さです。
Ken is ( as ) ( tall ) ( as ) Tom.
**正解**: as, tall, as

## 2. 読み上げ原稿
第1問！
カッコに入る単語は分かるかな？

### パターンA：具体例から入る
まずはこの文を見て。
"Ken is tall."
「ケンは背が高い」だね。
これに「トムと同じくらい」っていう情報を足したいとき、
tall を as と as で挟むんだ。
"Ken is as tall as Tom."
後ろの as Tom は「トムと同じくらい」っていう基準を表しているよ。
「誰と」同じなのかを後ろの as の後に置くんだ。
これで「ケン＝トム」っていう同じくらいの高さだよって意味になるよ。

### パターンB：型（ルール）から入る
「同じくらい」のルールはこれ！
「as + 形容詞 + as」
サンドイッチみたいに、言いたい言葉（tall）を as で挟むんだ。
as tall as Tom で「トムと同じくらい高い」。
この「as ... as」の型を使おう。

### パターンC：イメージ（気持ち）から入る
as ... as は「イコール（＝）」のイメージ。
天秤が釣り合っている状態だよ。
Ken と Tom の背の高さが、ピタッと釣り合っている感じ。
「どっちも同じだよ！」って伝えたいときに使うんだ。

### パターンD：NG例（間違い探し）から入る
よく "as taller as" って書いちゃうことない？
as と as の間は、そのままの形（原級）を使うルールなんだ。
er をつけたりしないで、そのまま tall を入れてね。
「挟むときは、そのまま！」って覚えよう。

### パターンE：リズム・暗記から入る
「同じくらい」は「as ... as」！
as tall as, as old as, as big as...
「アズ・アズ」で挟む！
リズムよく口ずさんで覚えちゃおう。

最後に読んでみよう！
Ken is as tall as Tom.
Ken is as tall as Tom.
素晴らしい！

## 3. 必要な画像
* ケンとトムが背比べをしていて、全く同じ高さのイラスト
* tall が as と as に挟まれるアニメーション
* 「as + 原級 + as」という枠組みのイラスト
* 天秤が釣り合っている（イコールの）イメージ
* "as taller as" にバツがついているイラスト
