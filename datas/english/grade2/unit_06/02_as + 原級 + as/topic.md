# as + 原級 + as の詳細ポイント

- as ... as (A is as ... as B): 「AはBと同じくらい〜だ」
-   - ... には形容詞・副詞の原級 (もとの形) が入る
-   - 否定文 (not as ... as): 「AはBほど〜ではない」
-   - 例文: He is as tall as I am. (彼は私と同じくらいの背の高さだ)
-   - 例文: This book is not as interesting as that one. (この本はあの本ほど面白くない)
-   - 倍数表現: twice as ... as (〜の2倍...)
-   - 慣用表現: as soon as possible (できるだけ早く)
