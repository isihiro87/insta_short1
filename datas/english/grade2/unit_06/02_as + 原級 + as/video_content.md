# as + 原級 + as 動画コンテンツ

## 1. 最初に提示する例題
**問題**: 日本語に合うように空所に適切な語を入れてみよう。
1. Ken is ( ) tall ( ) Tom. (ケンはトムと同じくらいの背の高さです)
2. This book is ( ) ( ) interesting ( ) that one. (この本はあの本ほど面白くありません)

**解答**:
1. as, as
2. not, as, as

## 2. 読み上げ原稿
第1問！
「ケンはトムと同じくらいの背の高さです」のように、
「AはBと同じくらい〜だ」って言いたいときは、
「as ＋ 形容詞や副詞（tall や fast など）の原級 ＋ as」を使うんだ。
原級っていうのは、
er や est がついていない、
もとの形のことだね。
「ケンはトムと同じくらいの背の高さです」なら
"Ken is as tall as Tom." になるよ。

これを否定文にして「not as ... as」とすると、
「AはBほど〜ではない」っていう意味になるんだ。
単に「同じではない」っていうだけじゃなくて、
「Bの方が程度が高い」っていうニュアンスが含まれるよ。
「この本はあの本ほど面白くない」なら
"This book is not as interesting as that one." となって、
あの本の方が面白い、
っていう意味になるんだね。

## 3. 必要な画像
1. **ケンとトム**: 同じくらいの身長の二人の男の子が並んでいるイラスト。
2. **2冊の本**: 一方はキラキラしていて面白そう（that one）、もう一方は少し地味（This book）なイラスト。

## 4. スライドに表示する内容
- **同等比較**: A is as ... as B. (AはBと同じくらい〜だ)
  - Ken is **as tall as** Tom.
- **否定の比較**: A is not as ... as B. (AはBほど〜ではない)
  - This book is **not as interesting as** that one.
