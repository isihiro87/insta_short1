# 「どちらが好きですか」 動画コンテンツ

## 1. 最初に提示する例題
**(1) 日本語訳に合うように、空所に適切な語を入れよう。**
犬と猫、どちらが好きですか。
(　) (　) you (　) (　), dogs or cats?
**正解**: Which do like better

## 2. 読み上げ原稿
**【パターンB：型（ルール）重視】**

問題！
カッコに入る単語は何だろう？
英語の文を作るときは、まず「型」を覚えておこう。

「どちらが好きですか」は、
Which do you like better, A or B? の型を使うよ。
「AとB、どっちが好き？」
っていう決まり文句だね。
比較級の better を like の後ろに置くのがポイント！

ここでもう一つ、大事なポイント！
「dogs, cats」って複数形になっているよね？
これは「犬全体」「猫全体」を指しているからなんだ。
「犬という生き物」について話すときは、
s をつけて複数形にするのがルールだよ。

「Which do you like better?」
一息で言えるように練習しよう！
