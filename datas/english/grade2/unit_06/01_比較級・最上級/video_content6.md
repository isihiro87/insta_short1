# 最上級 (不規則変化) 動画コンテンツ

## 1. 最初に提示する例題
**(1) 日本語訳に合うように、空所に適切な語を入れよう。**
私は全ての季節の中で夏が一番好きです。
I (　) summer (　) (　) of all seasons.
**正解**: like the best

## 2. 読み上げ原稿
**【パターンB：型（ルール）重視】**
問題！
カッコに入る単語は何だろう？
まず「型」を覚えておこう。

「～が一番好き」の型は、
like ～ the bestだよ。
「〜の中で」の部分は、今回みたいに「全ての」がついたら of を使うことが多い。

**(背景と範囲)**
なんで best を使うかっていうと、これは well（よく）の最上級だからなんだ。
good/well -> better -> best の変化はとても重要だ！

like ～ the best of/in ... に当てはめると、
"I like summer the best of all seasons." になるんだ。
