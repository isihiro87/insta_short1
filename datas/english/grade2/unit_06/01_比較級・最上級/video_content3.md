# 比較級 (長い単語) 動画コンテンツ

## 1. 最初に提示する例題
**(1) 日本語訳に合うように、空所に適切な語を入れよう。**
この本はあの本よりも面白いです。
This book is (　) (　) than that book.
**正解**: more interesting

## 2. 読み上げ原稿
**【パターンA：具体例重視】**

問題！
カッコに入る単語は分かるかな？

「面白い」は interesting だよね。
「より面白い」だから er をつけたい…
ところだけど、ちょっと待って！
interesting は長い単語だから、
er はつけられないんだ。

なんで er をつけないのかっていうと、
単語が長すぎて言いにくいからだよ。
interesting-er なんて噛んじゃうよね？だから代わりに、
前に more（もっと）を置いてあげるんだ。
beautiful（美しい）や popular（人気のある）みたいな長い単語は、
みんなこの more を使うよ。

だから、interesting の前に more を置いて、
more interesting にするよ。
これでバッチリ！
だから「この本はあの本よりも面白いです」は、
"This book is more interesting than that book." だね。
