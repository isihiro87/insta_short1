# NEW HORIZON 学年別 英文法一覧

## 中学1年生 (NEW HORIZON 1)

| Unit | 文法項目 | 概要・教科書の記述 |
| :--- | :--- | :--- |
| **Unit 1** | 一般動詞とbe動詞 | I am ..., I like ... などの一般動詞の文 / 疑問文・否定文 |
| **Unit 2** | 代名詞とcan | 主語に代名詞を使うbe動詞の文 / can を使う文 / this, that |
| **Unit 3** | 疑問詞 (1) | what, who, how, where, when を使う疑問文 |
| **Unit 4** | 複数形・命令文 | 名詞の複数形 / how many / what time / 命令文・否定の命令文 |
| **Unit 5** | 三人称単数現在形 | 三単現のs (肯定・疑問・否定) |
| **Unit 6** | 代名詞 (2)・疑問詞 (2) | 目的語に使う代名詞 (him, her等) / whose, which を使う疑問文 |
| **Unit 7** | 現在進行形・感嘆文 | 現在進行形 (肯定・疑問) / 感嘆文 (What/How) |
| **Unit 8** | 不定詞 (1)・look | 不定詞 (名詞的用法) / look + 形容詞 (〜に見える) |
| **Unit 9** | 一般動詞の過去形 | 規則動詞・不規則動詞 / 過去の疑問文・否定文 |
| **Unit 10** | be動詞の過去・過去進行形 | be動詞の過去形 / 過去進行形 / There is[are] ... の文 |

## 中学2年生 (NEW HORIZON 2)

| Unit | 文法項目 | 概要・教科書の記述 |
| :--- | :--- | :--- |
| **Unit 1** | 未来形・SVOO・SVOC | be going to / will / 目的語を2つとる動詞 (give等) / call+O+C |
| **Unit 2** | 接続詞 | when, if, because, that (〜ということ) |
| **Unit 3** | 不定詞 (2)・It is to | 不定詞 (副詞的用法・形容詞的用法) / It is ... + to + 動詞の原形 |
| **Unit 4** | 助動詞 (義務)・動名詞 | have to / must / 動名詞 (〜すること) |
| **Unit 5** | 疑問詞+to・接続詞that (2) | 疑問詞+不定詞 (how to等) / S+V+O+疑問詞+to / S+be+形容詞+that |
| **Unit 6** | 比較 | 比較級・最上級 / as + 原級 + as |
| **Unit 7** | 受動態 (受け身) | 受け身 (肯定・疑問) / byつきの受け身 / 助動詞つきの受け身 |
| **LR 3** | その他の重要表現 | come true / something + 形容詞 |

## 中学3年生 (NEW HORIZON 3)

| Unit | 文法項目 | 概要・教科書の記述 |
| :--- | :--- | :--- |
| **Unit 1** | 現在完了 (1)・SVOC・SVOO | 現在完了形 (経験) / make + O + 形容詞 / SVOO (that節) |
| **Unit 2** | 現在完了 (2) | 現在完了形 (完了・継続) / 現在完了進行形 |
| **Unit 3** | 不定詞 (3)・原形不定詞 | It is ... (for+人) + to / want+人+to / let[help]+人+動詞の原形 |

| **LR 3** | 構文復習 | so ... that ... (とても〜なので...) |
