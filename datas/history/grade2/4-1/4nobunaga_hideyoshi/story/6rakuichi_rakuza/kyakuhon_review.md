# 脚本添削レポート

## 元の脚本

```
N1: 信長が登場する前、商売には 2つの大きな壁があり、
一つ目、特定の商人だけが商売できる「座」という組合があった
二つ目、道の途中で通行料を支払うため商品の値段が高くなっていた
N2: 自由に商売ができず、経済が発展しにくい状態に対して信長は
安土城の城下町で楽市・楽座を実施
誰でも自由に市場で商売してOKに
「座」の特権を廃止
さらに、領内の関所を廃止
N3: 「自由に商売できる仕組み」を作って経済を発展させ、その力で天下統一を進めた
```

## 改善案

### N2（4行目）: 文字数超過・固い表現

**現在:**
```
自由に商売ができず、経済が発展しにくい状態に対して信長は（28字）
```

**改善案:**
```
こうした不便な仕組みを変えるため、信長は（20字）
```

**理由:**
- 「経済が発展しにくい状態」は中学生には抽象的
- 「不便な仕組み」のほうが直感的に理解できる
- 25字以内に収まる

---

### N3（9行目）: 文字数大幅超過・情報詰め込みすぎ

**現在:**
```
「自由に商売できる仕組み」を作って経済を発展させ、その力で天下統一を進めた（37字）
```

**改善案（2行に分割）:**
```
N3: こうして商売が活発になり、町は大いに栄えた（22字）
その力を武器に、信長は天下統一を進めていった（22字）
```

**理由:**
- 1文に2つの情報（経済発展＋天下統一）を詰め込んでいる
- 分割してそれぞれを明確に伝える
- 「町は大いに栄えた」で経済発展を具体的にイメージさせる

---

### 3行目: より具体的に

**現在:**
```
二つ目、道の途中で通行料を支払うため商品の値段が高くなっていた（30字）
```

**改善案:**
```
二つ目、道の関所で通行料を取られ、商品の値段が上がった（27字）
```

**理由:**
- 「道の途中で」より「道の関所で」のほうが具体的（8行目の関所廃止と対応）
- 「高くなっていた」→「上がった」でシンプルに
