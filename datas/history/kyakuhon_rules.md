# kyakuhon.txt 作成ルール

脚本ファイル（kyakuhon.txt）を作成・更新する際のルールです。

---

## 1. ファイル形式

- ファイル名: `kyakuhon.txt`
- 文字コード: UTF-8
- 配置場所: `datas/history/[学年]/[章]/[トピック]/story/story[連番]_[内容名]/`

### フォルダ命名規則

storyフォルダ名は以下の形式に従う:

```
[連番]_[内容を表す簡潔な名前]
```

**ルール**:
- 連番: 1から始まる通し番号
- 内容名: 英語またはローマ字で簡潔に（スペースは使わずアンダースコア `_` で区切る）

**例**:
| フォルダ名 | 内容 |
|-----------|------|
| `1_okehazama` | 桶狭間の戦い |
| `2_ashikaga_yoshiaki` | 足利義昭の将軍就任 |
| `3_hieizan` | 比叡山焼き討ち |
| `4_ikko_ikki` | 一向一揆の制圧 |
| `5_rakuichi_rakuza` | 楽市・楽座 |
| `6_honnoji` | 本能寺の変 |

## 2. 構造（N1/N2/N3 形式）

脚本は3部構成で記述する。

```
N1: [導入行1]
[導入行2（オプション）]
[導入行3（オプション）]
N2: [展開行1]
[展開行2]
[展開行3]
[展開行4（オプション）]
N3: [結論行1]
[結論行2（オプション）]
```

### 各セクションの役割

| セクション | 役割 | 内容例 |
|-----------|------|--------|
| **N1（導入）** | 背景・状況設定 | 「信長が登場する前、商売には2つの大きな壁があり...」 |
| **N2（展開）** | 具体的な出来事・行動 | 「信長は安土城の城下町で楽市・楽座を実施」 |
| **N3（結論）** | 結果・歴史的意義 | 「経済を発展させ、その力で天下統一を進めた」 |

### 構造ルール

- N1/N2/N3の順序を守る
- 各セクションは1行以上、4行以下
- 合計8-10行程度
- 1行は50文字以内が目安（読み上げ時間を意識）

---

## 3. 口調・文体（説明調）

**基本方針**: 客観的で分かりやすい説明調に統一する。

### 推奨する表現

| パターン | 例 |
|---------|-----|
| 事実の説明 | 「〜があった」「〜を実施」「〜した」 |
| 過去形 | 「〜していた」「〜になった」 |
| 状況説明 | 「当時〜」「このとき〜」 |
| 因果関係 | 「〜ため」「〜により」「〜に対して」 |

### 避ける表現

| NG | 理由 |
|----|------|
| 「〜でしょう」「〜かもしれない」 | 曖昧な表現 |
| 過度な感嘆詞（「なんと！」「すごい！」） | 感情的すぎる |
| 口語的すぎる表現（「〜っていうのは」） | 口語調ではない |

### 用語の扱い

- **固有名詞**: 正確に記述（人物名、地名、事件名）
- **専門用語**: 必要に応じて補足説明を添える
  - 例: 「『座』という組合」「浄土真宗の信者たちが団結して起こした一向一揆」

## 4. 入力素材の形式（柔軟対応）

`/make-story` コマンドは以下の形式をすべて受け付ける。

### パターンA: 箇条書きリスト

```
- 信長は楽市・楽座を実施した
- 座の特権を廃止
- 関所も廃止した
```

### パターンB: 時系列メモ

```
1560年: 桶狭間の戦い
1568年: 上洋
1571年: 比叡山焼き討ち
```

### パターンC: 要点+詳細

```
【要点】楽市・楽座の実施
【詳細】商売の自由化により経済発展
【背景】座という組合が商売を独占
```

### パターンD: 散文・参考資料

```
織田信長は安土城の城下町で楽市・楽座を実施した。
これにより誰でも自由に商売ができるようになった...
```

## 5. サンプル

### 素材（material.md）

```
楽市・楽座について
- 信長以前：「座」という組合が商売を独占
- 信長以前：関所で通行料を取られ、商品価格が高騰
- 信長の政策：安土城下町で楽市・楽座を実施
- 結果：誰でも自由に商売OK、座の特権廃止、関所廃止
- 意義：経済発展により天下統一の資金力を確保
```

### 出力（kyakuhon.txt）

```
N1: 信長が登場する前、商売には 2つの大きな壁があり、
一つ目、特定の商人だけが商売できる「座」という組合があった
二つ目、道の途中で通行料を支払うため商品の値段が高くなっていた
N2: 自由に商売ができず、経済が発展しにくい状態に対して信長は
安土城の城下町で楽市・楽座を実施
誰でも自由に市場で商売してOKに
「座」の特権を廃止
さらに、領内の関所を廃止
N3: 「自由に商売できる仕組み」を作って経済を発展させ、その力で天下統一を進めた
```
