# テーマリサーチ

入力されたテーマについて、40秒ショート動画の脚本（kyakuhon.txt）を書くために必要な情報を `research.md` にまとめます。

## 引数

- `$ARGUMENTS`: テーマ（例: `桶狭間の戦い`、`本能寺の変`）

---

## フェーズ1: 情報収集

WebSearchで以下を収集:
- 基本情報（いつ・どこ・誰・何が起きた）
- 出来事の流れ（原因→経過→結果）
- 面白いポイント（意外な事実、ドラマチックな場面）

---

## フェーズ2: 保存先の確認

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
テーマ: [テーマ名]

保存先フォルダのパスを入力してください。
例: datas/history/grade2/4-1/4nobunaga_hideyoshi/story/9_new
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

---

## フェーズ3: research.md の作成

以下の形式で作成（**40秒動画用にコンパクトに**）:

```markdown
# [テーマ名]

## ざっくり説明
[3〜4文で出来事全体を説明]

## 流れ
1. [きっかけ]
2. [経過1]
3. [経過2]
4. [結果]

## 登場人物
- **[人物1]**: [一言説明]
- **[人物2]**: [一言説明]

## キーワード
- [語句1]: [意味]
- [語句2]: [意味]

## 動画のネタ
- [面白いポイント1]
- [面白いポイント2]
- [ドラマチックな見せ場]

## 脚本案

### 案1: [切り口]
N1: [導入 - 興味を引く一文]
本文: [展開 - 2〜3文]
N3: [結論 - 印象に残る一文]

### 案2: [別の切り口]
N1: [導入]
本文: [展開]
N3: [結論]

## 注意
- [諸説ある点など]
```

## フェーズ4: 完了報告

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
完了: [保存先]/research.md

次のステップ:
1. research.md を確認
2. kyakuhon.txt を作成
3. /review-story で添削
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

---

## 重要なルール

1. **40秒動画に収まる分量** - 詳しすぎる情報は不要
2. **脚本案を2つ以上** - 切り口の選択肢を提供
3. **中学生にわかりやすく** - 難しい言葉は避ける
4. **動画映えするポイント重視** - 教科書的な説明より面白さ
