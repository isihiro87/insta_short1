# rename-audio

audioフォルダ内のファイル名の先頭にある番号を後ろに移動し、0始まりを1始まりに変更するコマンド。

## 引数

$ARGUMENTS: audioフォルダのパス（複数指定可、カンマ区切り）

## 処理内容

1. 指定されたaudioフォルダ内のファイルを確認
2. `[番号]-[名前].[拡張子]` 形式のファイルを `[名前]-[番号+1].[拡張子]` にリネーム
3. 変更前後のファイル名を表示

## 実行手順

1. 各フォルダについて、現在のファイル一覧を表示
2. リネーム対象を確認（先頭が数字-で始まるファイル、または末尾が-数字のファイル）
3. bashのmvコマンドでリネーム実行
4. 結果を表示

## リネームルール

### 先頭に番号がある場合
- `0-muro-metsubou.wav` → `muro-metsubou-1.wav`
- `1-4ikko_ikki_nar.wav` → `nar-4ikko_ikki-2.wav`
- 先頭の `[数字]-` を抽出し、+1して拡張子の前に `-[数字+1]` として追加
- ファイル名に `_nar` や `_narr` が含まれる場合は、`nar-` を先頭に移動

### 末尾に番号がある場合（0始まり→1始まり）
- `muro-metsubou-0.wav` → `muro-metsubou-1.wav`
- `4ikko_ikki_nar-1.wav` → `nar-4ikko_ikki-2.wav`
- 末尾の番号を+1
- ファイル名に `_nar` や `_narr` が含まれる場合は、`nar-` を先頭に移動

## 注意

- ドライランで確認してからユーザーに実行確認を求める
- 番号の衝突を避けるため、大きい番号から順にリネームする
