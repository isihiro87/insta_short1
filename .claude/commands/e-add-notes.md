# 英単語補足追加

words_XX.mdファイルの各単語に、語源・覚え方のコツ・関連語・使い方などの補足を追加します。

## 引数

- `$ARGUMENTS`: wordsフォルダパス（例: `datas/english/grade3/unit_06/quiz/words`）

---

## ステップ1: ファイル確認

### 1.1 words_XX.mdファイルの検索

```bash
ls [フォルダパス]/words_*.md
```

### 1.2 各ファイルの単語を確認

ファイルごとに単語と日本語訳を一覧表示。

---

## ステップ2: 補足の生成

### 2.1 補足の基準

各単語に対して、以下の観点から**最も有用な1つ**を選び、**20〜30字**で補足を作成:

| 観点 | 例 |
|-----|-----|
| 語源・成り立ち | im-(中へ)+port(運ぶ)→輸入 |
| 覚え方のコツ | donateはdonation(寄付)の動詞形 |
| 関連語・派生語 | 名詞形はsurvival→survive(動) |
| 重要な使い方 | depend on〜で「〜に頼る」 |
| 注意点・間違いやすい点 | abroadは前置詞不要、go abroad |
| 対義語・類義語 | ⇔import(輸入)の反対はexport |

### 2.2 補足のルール

1. **1単語1補足**（複数の観点を混ぜない）
2. **20〜30字**（短すぎず長すぎず）
3. **中学生にわかりやすい表現**
4. **試験や実用で役立つ情報を優先**
5. **既に補足がある場合はスキップまたは上書き確認**

### 2.3 補足例

```
donate
･･･を寄付する
donationは「寄付」、donorは「寄付者」

supply
供給、必需品
動詞で「供給する」、school suppliesで「学用品」

abroad
外国に、海外に
go abroad、study abroadのように前置詞不要
```

## ステップ3: ファイル更新

### 3.1 更新形式

**更新前**:
```
単語
日本語訳

次の単語
...
```

**更新後**:
```
単語
日本語訳
補足（20〜30字）

次の単語
...
```

### 3.2 各words_XX.mdを更新

既存の補足がある場合は保持し、ない単語のみ追加。

## 注意事項

- 補足は動画では表示されない（学習用メモとして保持）
- qa.mdには補足は含まれない（英単語と日本語訳のみ）
- 補足を動画に表示したい場合は別途対応が必要
